# Precious Metal Rates - Svacron

Live rates for Gold, Silver, and Platinum prices in India.

## Features

- **Real-time Rates**: Live prices for Gold, Silver, and Platinum
- **Multiple Purities**: Display rates for different purity levels
- **Historical Data**: One month price history with change indicators
- **Interactive Charts**: Yearly data visualization (1Y, 3Y, 5Y, 10Y, All Time)
- **Firebase Backend**: Data fetched from Firebase Realtime Database
- **SEO Optimized**: Full metadata and schema markup

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3002](http://localhost:3002)
